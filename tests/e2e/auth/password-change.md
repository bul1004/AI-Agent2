### パスワード変更 (`password-change`)

- メールサインアップ後のパスワード変更機能が正しく動作することを確認

#### テストケース

| テスト名                                                           | 確認事項                                               |
| ------------------------------------------------------------------ | ------------------------------------------------------ |
| `メールサインアップ後、設定からパスワードを変更できる`             | 正しい現在のパスワードで新しいパスワードに変更できる   |
| `現在のパスワードが間違っている場合はエラーが表示される`           | 間違ったパスワードでエラートーストが表示される         |
| `新しいパスワードが8文字未満の場合はバリデーションエラーが表示される` | 8文字未満のパスワードでバリデーションエラーが表示される |

#### 手順（パスワード変更 - ハッピーパス）

1. テストユーザーを新規作成してログイン状態にする
2. アバターボタンにホバーしてプロフィールメニューを開く
3. アカウント設定をクリックして設定モーダルを開く
4. プロフィール編集ボタンをクリック
5. パスワード設定ボタンをクリック
6. 現在のパスワードを入力
7. 新しいパスワードを入力
8. 新しいパスワード（確認）を入力
9. 変更ボタンをクリック
10. 成功トースト「パスワードを変更しました」が表示されることを確認

#### 手順（現在のパスワード間違い）

1. テストユーザーを新規作成してログイン状態にする
2. パスワード設定画面を開く（上記手順 2-5）
3. 間違った現在のパスワードを入力
4. 新しいパスワードを入力
5. 変更ボタンをクリック
6. エラートーストが表示されることを確認

#### 手順（バリデーションエラー）

1. テストユーザーを新規作成してログイン状態にする
2. パスワード設定画面を開く（上記手順 2-5）
3. 正しい現在のパスワードを入力
4. 8文字未満の新しいパスワード（例: `short`）を入力
5. 変更ボタンをクリック
6. 「パスワードは8文字以上で入力してください」エラーが表示されることを確認

#### 前提条件

- なし（テストごとに新規ユーザーを作成）

#### 使用する data-testid

| セレクタ                         | 要素                       |
| -------------------------------- | -------------------------- |
| `profile-avatar-button`          | プロフィールアバターボタン |
| `profile-menu-account`           | アカウント設定メニュー項目 |
| `edit-profile-button`            | プロフィール編集ボタン     |
| `password-settings-button`       | パスワード設定ボタン       |
| `current-password-input`         | 現在のパスワード入力欄     |
| `new-password-input`             | 新しいパスワード入力欄     |
| `confirm-password-input`         | パスワード確認入力欄       |
| `change-password-submit`         | パスワード変更ボタン       |
